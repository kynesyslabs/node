# HTTP Requests

Demos accepts HTTP requests on the "/" endpoint, exposed by the src/libs/network/server_rpc.ts file.

The listening port is defined at runtime as per src/index.ts.

## Request format

A request is a JSON object with the following structure:

```json
{
    method: string
    params: any[]
}
```

And must contain the following headers:

- `identity`: The identity of the sender, expressed as an hexadecimal string
- `signature`: The signature of the sender, expressed as an hexadecimal string

The signature is generated by signing the `identity` string with the sender's private key.

## Response format

A response is a JSON object with the following structure:

```json
{
    result: number // HTTP status code
    response: any
    require_reply: boolean
    extra: any
}
```
