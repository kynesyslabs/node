# Kyntainer

## DEMOS on Docker

### Prerequisite

You should have either a copy of the node's software into node_base subfolder or be in the docker subfolder of the above mentioned repository.

### Preparation

If you are in the node's software repository under the docker subfolder, just run `prepare_docker_environment.sh`.
It will copy the necessary files in the appropriate folder.

Then, you should look at the three yaml files to understand or to customize your experience.

### Adding a node

Generate a .demos_identity, copy (or create) an empty data/chain.db and replicate the examples.
